# Palette

**uhgenie-ui**에서 사용하는 팔레트입니다. 팔레트를 사용하면 상황에 맞게 구성 요소의 색상을 수정할 수 있습니다.

## 사용 방법

<Source language="js" dark code={`import { primary, success } from 'uhgenie-ui';`} />

## 색상 목록

<ColorPalette>
  {Object.entries(palette).map(([paletteName, color]) => (
    <ColorItem
      key={paletteName}
      title={paletteName}
      subtitle=""
      colors={{ [paletteName]: color }}
    />
  ))}
</ColorPalette>
